1️⃣ IBD RS (IBD Relative Strength Rating)

무엇인가: IBD가 모든 미국 주식을 12개월 가격 성과로 순위 매긴 백분위. 99면 상위 1%, 50이면 중간.
특징: 미국 전체 주식 대비 상대 강도를 단일 숫자로 보여주는 검증된 기준. 하지만 3개월 이상의 성과가 누적되어야 반영되어 느리다.
사용처: "이 종목이 최근 1년간 시장 평균보다 얼마나 강했나?" → 장기 강세 검증, 스크리너 1차 필터 (80+, 90+)
한계: 이미 올라간 종목만 발굴 → 선행 매수 기회 놓침. 최근 폭등한 종목이 아직 IBD RS에 반영 안 될 수 있음.

2️⃣ RS Line (Relative Strength Line)

무엇인가: 종목 종가를 SPY 종가로 나눈 원시 비율. 예: 종목 $100, SPY $450 → RS Line = 0.222.
특징: 절대값은 의미 없고, 시간에 따른 변화가 중요. 상승하면 SPY 대비 강해지는 중, 하락하면 약해지는 중. 매일 즉시 변동.
사용처: RS Line의 52주 신고가 갱신 여부 판단. O'Neil 원전의 "RS Line이 주가보다 먼저 신고가를 찍는다"는 선행 신호의 기준.
신고가 판단: 오늘 RS Line 값이 지난 52주 내 모든 값보다 높으면 = RS Line 신고가 (true flag)
→ 이 시스템의 "RS Line선행" / "RS Line신고" 필터가 사용하는 기준

3️⃣ RS now (RS Line 52주 백분위 위치)

무엇인가: RS Line이 자신의 52주 범위 내에서 어디쯤 위치하는지를 1~99로 변환한 값.
직관적 이해: RS Line이 52주 최고점 = 99, 52주 최저점 = 1, 중간 = 50.
특징: IBD RS의 선행 버전. IBD RS가 느리게 반영하는 강세를 실시간으로 포착. 같은 값 범위(1~99)라 IBD RS와 직접 비교 가능.
사용처: 정렬 및 시각 비교용. 신고가 플래그로는 사용되지 않음 (백분위라 "신고가" 개념 자체가 없음 — 99가 최대값).
갭 = RS now − IBD RS: 양수면 IBD RS가 아직 반영하지 못한 신규 부상 중, 음수면 약화 중.
💡 RS now를 사용하기에 적절한 경우
✓ 신규 부상 종목 발굴 — IBD RS가 85인데 RS now가 99면, 최근 급격한 강세로 곧 IBD RS도 올라올 후보. "갭 양수" 종목 찾기.
✓ 종목 간 현재 강도 비교 — 같은 IBD RS 92인 두 종목이 있을 때, RS now 98 vs 70이면 전자가 지금 이 순간 더 강함.
✓ 섹터 로테이션 포착 — 섹터 평균 RS now가 평균 IBD RS보다 크게 높으면 신규 부상 섹터 (갭 상위 섹터).
✓ 정렬 기준 — "현재 강세" 순으로 종목을 훑어보고 싶을 때 RS now 내림차순 정렬.
✓ 약화 신호 조기 감지 — IBD RS는 95인데 RS now가 50으로 떨어졌다면, RS Line이 52주 범위의 중간까지 내려온 상태 → 곧 IBD RS도 떨어질 것.
✓ 보유 종목 모니터링 — 보유 중 RS now가 90→70→50으로 떨어지는 추세는 추세 약화 경고.
⚠ RS now 사용 시 주의사항
✗ "RS now 99 = 신고가" 아님 — 99는 52주 내 최고 위치일 뿐. 실제 신고가 갱신 여부는 RS Line 플래그로 판단.
✗ 인덱스 ETF에서 왜곡 — VOO/SPY 등은 RS Line 변동폭이 극히 작아 RS now가 무의미. 개별 종목에만 사용.
✗ 거래량 극소 종목 왜곡 — 일일 변동이 커서 RS Line이 출렁 → RS now가 실제 강도를 반영 못 할 수 있음.
✗ RS now 혼자만 보고 매수 금지 — RS now는 "현재 위치"일 뿐. 반드시 RS Line 신고가, 패턴, Stage, 거래량과 결합해 판단.
✗ IBD RS 대체재가 아님 — RS now는 IBD RS를 보완하는 지표. IBD RS가 낮으면(70 미만) RS now가 99여도 장기 강세는 검증되지 않은 것.

🎯 세 지표의 이상적 조합 (최강 신호)

가장 강력한 매수 후보:
① IBD RS ≥ 85 (장기 검증된 강세)
② RS now ≥ 90 (현재도 강세 유지)
③ 갭(RS now − IBD RS) 양수 (부상 중)
④ RS Line 선행 신고가 ✓ (선행 신호)
⑤ 주가는 아직 52주 신고가 전 (진입 여력)

이 5가지가 모두 충족되면 "이미 강하고 + 지금도 강해지고 있으며 + 진입 여력도 남은" 완벽한 RS 프로파일. 실전에서는 하루에 몇 종목 안 나올 정도로 드물지만, 나오면 높은 확률의 후보.

📌 실전 사용 가이드

스크리너 탭: IBD RS 필터(80+, 91+)로 1차 압축 → RS now 컬럼 정렬로 현재 강세 확인 → RS Line선행 필터로 진입 기회 포착
RS 모멘텀 탭: 갭(RS now − IBD RS) 내림차순 정렬 → 신규 부상 종목 발굴 → Stage 2 필터로 추세 확인
섹터 분석 탭: 갭 정렬로 신규 부상 섹터 파악 → 해당 섹터 종목들을 스크리너 탭에서 확인
종목 모달 차트: IBD RS Line(파란선)과 RS now Line(초록선) 두 라인을 시각적으로 비교 → 발산/수렴 패턴 확인
핵심 논리
RS Line이 먼저 움직이고, 주가가 뒤따라온다. 기관 매집의 선행 지표를 활용해 돌파 전 진입 기회를 포착한다.
RS선행 필터
주가는 아직 신고가 전, RS Line은 먼저 신고가. 진입 기회가 남아 있는 후보 발굴용.
⚡가속+↗RS Line신고 필터
RS Line 가속 중 + RS Line 신고가 동시. 돌파 임박 타이밍 확인용.

Step 1 — 종목 발굴 (매일 5분)

1-1. RS선행 필터로 후보 리스트업

스크리너 탭에서 다음 순서로 필터 설정:

  1. RS: 91~99 (최강 종목 우선)
  2. Stage: S2 (이미 상승 추세 진입)
  3. 신호: RS Line선행

여기서 나오는 종목들이 후보군입니다. RS Line은 신고가를 찍었지만 주가는 아직인 상태라, 앞으로 1~4주 내 주가도 따라올 가능성이 높습니다.

1-2. 추가 검증 조건 (종목 클릭 후 모달에서 확인)

  • 거래량 확인 V 표시: 있어야 매집 중 (없으면 제외)
  • 피벗까지 거리: 양수 + 10% 이내면 좋음
  • Pass 8/8: Minervini Trend Template 완벽 통과면 최우선
  • 주봉 차트: 10주·30주 MA 정배열 확인
  • 베이스 형태: 일봉 차트에 베이스 존(연파랑) + 피벗선(점선) 확인

1-3. 워치리스트 등록

조건을 만족하는 종목은 ★ 클릭하여 워치리스트에 추가. 보통 하루에 3~10개가 적절합니다. 너무 많으면 관리가 안 되고, 0~2개면 시장이 약세장일 가능성.

Step 2 — 매수 타이밍 대기 (1~4주)

2-1. 매일 아침 루틴

  1. run_daily.bat 실행
  2. RS 모멘텀 탭 → ⚡가속+↗RS Line신고 필터 클릭
  3. 워치리스트(★) 종목 중 이 필터에 걸린 게 있는지 확인

이 필터에 걸렸다면 임박 신호입니다. RS가 가속 중이면서 RS신고가까지 찍었다는 뜻이라, 주가 돌파가 임박한 상태.

2-2. 매수 트리거 조건 (모두 충족 시)

  1. 돌파 가격 도달: 현재가 ≥ 피벗 가격
  2. 돌파 거래량: 50일 평균의 1.4배 이상 (V 표시)
  3. Stage 2 유지: Pass 7/8 이상
  4. 시장 전체 방향: SPY/QQQ가 50일 MA 위 (약세장이면 매수 금지)

2-3. 분할 매수 (Progressive Exposure)

한 번에 풀 포지션 X. 다음처럼 나눠서 진입:

1차 50%
피벗 돌파 + 거래량 확인
2차 30%
1차가 +3% 도달
3차 20%
1차가 +5~7% 도달

Minervini의 "Progressive Exposure". 잘못된 돌파 손실 제한 + 진짜 돌파 수익 극대화.

Step 3 — 포지션 관리

3-1. 손절선 설정 (매수 즉시)

1차 매수가 기준 -7% 손절선 (O'Neil 원칙)
실제로는 -5%에서 끊는 게 더 안전 (Minervini는 -3~5% 권장). 7%는 절대 초과 금지. 손절은 무조건 지킨다.

3-2. 진행 관리

매일 RS 모멘텀 탭에서 보유 종목 체크:

  • RISING 유지 → 홀드
  • FLAT 전환 → 경계, 차트 확인
  • FALLING 전환 → 이익 실현 고려
  • 주가/RS Line 신고가 H 표시 유지 여부

3-3. 이익 실현 기준

부분 익절 지점:

  • +20~25%: 1/3 익절. 단, 3주 이내 20% 달성 시 최소 8주 보유 (빅 위너 가능성)
  • +50%: 1/3 추가 익절
  • 나머지 1/3: 추세 추종 (50일 MA 이탈 시 매도)

전량 매도 신호:

  • 50일 MA 종가 기준 명확한 이탈
  • 주봉 기준 30주 MA 이탈
  • RS 모멘텀 탭에서 FALLING + 추세 전환
  • 섹터 분석에서 해당 섹터 갭이 음수로 전환

Step 4 — 포트폴리오 관리

  • 종목당 최대 8~12% (총자본 대비)
  • 동시 보유 5~8개. 강세장 6~8개 / 약세장 2~3개 또는 전액 현금
  • 섹터 분산: 섹터 분석 탭에서 갭 양수 상위 3~5 섹터에 집중. 동일 섹터 3종목 이상 금지
  • 시장 방향 체크: SPY가 200일 MA 이탈 시 전량 매도 고려

Step 5 — 주간 리뷰 (토요일 30분)

  1. 이번 주 매수 종목 재검토 (감정 매수 아니었나)
  2. 패턴 감지 탭 → 4+ 전체 버튼으로 다음 주 후보 발굴
  3. 섹터 분석 변화 확인 (섹터 로테이션 파악)
  4. 손절 종목 분석: 왜 실패했는지 기록
  5. 승리 종목 분석: 어떤 조건이 가장 강했는지 기록

  • 감정 거래 금지: "조금만 더 오르면...", "회복할 거야..."는 계좌를 망친다
  • 백테스트는 참고용: 과거 승률이 미래를 보장하지 않음. 시장 환경이 바뀌면 패턴 승률도 바뀜

📊 5가지 개별 지표 해설

① 지수 Stage (3개 지수)
추적 대상: S&P 500 (전체시장) · NASDAQ (성장주) · Russell 2000 (소형주 — risk-on 리트머스 ★)
계산: 21/50/200 MA 정배열 + 현재가 위치로 Stage 1~4 판정 (Weinstein/Minervini 기준 간소화)

🎯 핵심 팁: Russell 2000은 특별히 중요합니다. Russell이 S&P보다 약하면 시장이 '안전자산 로테이션' 모드라는 신호 — 작은 위험 회피 기류가 먼저 감지됩니다. Russell 50MA 아래 이탈은 Under Pressure 자동 전환의 핵심 조건.
② Follow-Through Day (FTD) — 오닐의 바닥 포착
정의: Correction 이후 반등 시도 중 특정 조건 만족 시 "시장 바닥 확정" 신호.

조건 (모두 충족):
· 저점 이후 반등 4일차 이상 (d4~d7 핵심 구간)
· 당일 종가 +1.25% 이상 (S&P) / +1.5% 이상 (NASDAQ)
· 거래량 전일 대비 증가
· 이후 반등 저점을 다시 깨지 않음 (깨지면 FTD 무효화)

💎 왜 d4 이상인가?
2~3일차 반등은 약한 손의 단기 매수에 의한 노이즈가 많습니다. 4일차부터는 실제 기관 매수 참여가 확인된 진짜 반등일 가능성이 높아집니다. O'Neil의 80년 데이터 분석 결과.
③ Distribution Day (DD) — 기관 매도 카운터
정의: 지수가 -0.2% 이상 하락 + 거래량 전일 대비 증가한 날. 기관이 물량을 털고 있다는 증거.

IBD 공식 25거래일 윈도우:
DD 수상태
0~4개🟢 Healthy Uptrend
5개🟡 주의 신호
6~7개🟠 Under Pressure 전환
8개+🔴 Correction 임박/진행

DD 만료 규칙 2가지 (IBD 공식):
· 25거래일 경과 → 자동 만료
· 해당 DD 종가 대비 6% 반등 → 무효화 (강한 회복 증거)
→ 대시보드에 "✓ 6% 반등으로 만료된 DD: N개" 녹색 박스로 표시.
④ Market Breadth — 시장 폭 ($S5FI / $NDFI)
정의: 지수 구성 종목 중 50MA 위에 있는 비율(%).

3개 시리즈 비교:
· 🔵 S&P 500 > 50MA — 전체 시장 건강도
· 🟠 NASDAQ 100 > 50MA — 기술주·성장주 건강도
· 🟢 시스템 500+ > 50MA — IBD 관심 종목 건강도 (가장 높은 값 — 선별된 종목)

건강도 구간:
· 70%+ 강세 전방위 확산 · 55~70% 건강 · 40~55% 중립 · 25~40% 약화 · 25%↓ 위험

⚠ 다이버전스 = 가장 중요한 경고 신호
지수는 계속 신고가 갱신 중인데 이 라인이 하락하면 → "소수 대형주만 상승 중 (Mag 7 효과)" → 상승 동력 좁아짐 → 조정 임박 경고. 2022년 초 · 2020년 2월 등 과거 주요 조정 전에 모두 나타났던 패턴.

반대로 지수는 횡보지만 이 라인이 상승하면 → 바닥 회복 reversal 신호.

🏠 자체 계산 방식: 시스템이 이미 다운로드한 500+ 종목 × Wikipedia의 S&P 500/NASDAQ 100 구성 종목 매칭. Yahoo/Barchart 외부 API 의존 제로, rate limit 없음.
⑤ 이동평균선 매트릭스 (각 지수 × 3개 MA)
각 지수(S&P · NASDAQ · Russell) × 3개 MA(21 · 50 · 200) = 9개 체크박스.

Stage 자동 판정:
· Stage 2: 21>50>200 완전 정배열 + 현재가 MA 위 = 상승 추세
· Stage 2 초기: 50, 200 위지만 21 불안정 = 추세 전환 중
· Stage 3: 200MA 위지만 50MA 아래 = 고점권 조정
· Stage 4: 200MA 아래 = 하락 추세 (매수 금지)

🎯 실전 체크: 9/9 모두 체크되면 완벽한 강세. 6/9 이하면 Under Pressure 가능성. 3/9 이하면 Correction.

❓ 자주 묻는 질문

Q. Distribution Day가 왜 25일인가? 20일 아닌가?
A. IBD 공식 기준이 25거래일 (약 5주)입니다. Nasdaq 공식 칼럼, TradingView IBD 인디케이터, useThinkScript 커뮤니티 등 모든 권위 있는 소스에서 25일로 명시. 20일(4주)은 너무 짧아 bear market 조기 진입의 오탐이 많습니다.
Q. Breadth가 왜 자체 계산인가? 공식 $S5FI 값이 더 정확하지 않나?
A. Yahoo Finance `^S5FI` 는 rate limit이 자주 걸려 불안정합니다. Barchart API는 키 발급·유료 tier 등 장애물이 많습니다. 자체 계산은 Wikipedia 구성 종목 리스트 × 우리가 이미 받는 가격 데이터로 산출 — 외부 API 의존 제로 + rate limit 제로. 공식 S5FI와 상관계수 0.95+ (커버리지 80%+ 기준).
Q. 오늘 FTD 발생했는데 어느 종목부터 사야 하나?
A. FTD 당일 ~ 첫 2주가 연 최대 알파 구간입니다. 순서는:
1순위 — 오늘 Phase 4 상향 전환 (NEW↑ 배지)
2순위 — Phase 4+ 종목 (장기배경 양호)
3순위 — 베이스 돌파 직후 (RS Line 선행 신고가)
Alt+6 (Phase 전환 프리셋) 활용.
Q. Russell 2000은 왜 그렇게 중요한가?
A. 소형주는 risk-on 심리의 리트머스입니다. 투자자들이 위험을 감내할 때만 소형주가 오릅니다. 반대로 위험 회피 모드 시작 시 가장 먼저 빠지는 자산. S&P는 Mag 7만으로도 끌려갈 수 있어 '속이는' 지표일 수 있지만, Russell은 속이지 못함. Russell 50MA 이탈 = 시장 내부가 무너지기 시작했다는 선제 경고.
Q. Correction 상태인데 Phase 4 종목이 계속 나온다. 사도 되나?
A. 절대 안 됩니다. O'Neil 통계: 약세장 돌파 성공률은 약 33% (강세장은 70%+). 즉 3번 중 2번은 실패. Correction에서는 아무리 좋은 종목도 시장과 함께 빠집니다. 시스템이 🔴 배지 + 빨간 경고 배너로 알려주는 이유가 이것.
Q. 매일 run_daily.bat 돌리지 않으면 어떻게 되나?
A. 그 날의 데이터가 공백이 되어:
· DD 카운트가 놓칠 수 있음 (경고 상태 늦게 인지)
· FTD 놓치면 연 최대 알파 구간 놓침
· Breadth 히스토리 단절로 차트가 끊김
· Regime 히스토리 타임라인에 회색 칸 (데이터 없음)
장이 열리는 날은 반드시 실행 권장.

🎯 "갭 전략" — 2지표 조합 4가지 시나리오

🎯 Phase & MTR 통합 가이드

이 시스템에 독창적으로 추가된 기능들의 완전 설명서 · v2.1
📍 왜 이게 필요한가?
기존 RS 지표 · Stage · 패턴은 각각 따로 작동했습니다. 예를 들어 "RS 가속 중"이라는 신호만 봐서는 이게 지금 사야 할 때인지, 팔아야 할 때인지를 판단하기 어려웠습니다.

이 가이드에 나오는 MTR State (0~7)와 Base Phase (0~7)는 "지금 이 종목이 생애주기 어디에 있는가"를 하나의 숫자로 압축해서 보여줍니다.

결론: 이제 Phase 숫자 하나만 봐도 관찰 / 매집 / 대기 / 진입 ★ / 홀드 / 익절 / 청산 중 어떤 액션을 해야 하는지 즉시 판단 가능합니다.

🔢 MTR State (Multi-Timeframe RS State)

한 줄 정의
1주·3주·6주 RS 변화율의 부호를 조합해서 0~7로 인코딩한 모멘텀 생애주기 코드.
계산 방법 (쉽게 설명)
종목의 RS Line이 최근 3구간에서 올랐는지 내렸는지만 봅니다:
c1 = 최근 1주 변화율 (현재 vs 1주 전)
c3 = 중기 변화율 (1주 전 vs 3주 전)
c6 = 장기 변화율 (3주 전 vs 6주 전)

각각 양수(+)면 1, 음수(−)면 0. 이 3비트를 묶으면 000~111 = 0~7.
예: c6=+, c3=+, c1=+ → 111 → State 7 (완전 상승)
8가지 State 전체 맵
State 부호 패턴 상태명 의미 & 직관
0 − − − 🔴 지속 하락 6주 내내 RS 약세. 추세 역전 없음. Stage 4 전형.
1 − − + 🟡 첫 반등 약세에서 최근 1주만 반등. 너무 이름 — 추세 확인 대기.
2 − + − ⚪ 헛된 회복 중간만 잠깐 올랐다가 다시 약세. 휩쏘 / 잡음.
3 − + + 🟢 전환 신호 ★ 장기 약세 → 중단기 모두 상승 전환. Stage 1→2 전환 전형. 진입 후보군.
4 + − − 🟠 최근 약세 장기 좋았는데 최근 빠짐. 추세 피로 또는 이탈 경고.
5 + − + 🟣 고르지 못한 상승 장기·단기는 좋은데 중간 쉰 적 있음. Stage 2 성숙 단계 흔함.
6 + + − 🟤 고점 풀백 장·중기 상승 유지, 최근 1주만 조정. 이익실현 검토 시점.
7 + + + 🟢 완전 상승 ★ 6주 내내 상승 지속. Stage 2 본격 리더. 이상적 홀드 상태.
💡 State 가운데 특히 중요한 2개
State 3 (− + +) = 전환 신호 — 바닥권에서 이제 막 돌아선 상태. 가장 이른 진입 후보.
State 7 (+ + +) = 완전 상승 — 이미 올라가는 중. 홀드 / 추가매수의 이상적 배경.

나머지 6개 State는 참고용이고, 실전에서는 3과 7 위주로 스캔하면 효율적입니다.
⚡ State "++" 강도 수식어
State 숫자 뒤에 "++" 가 붙으면 c1 > c3 > c6 (단조 가속) 을 의미합니다.
예: State 3++ = 전환 신호 + 가속 지속 → 최상의 조기 진입 타이밍.
예: State 7++ = 완전 상승 + 가속 지속 → 본격 리더 확신 강화.

대시보드에서는 "가속추세" 플래그(⚡⚡)가 이 ++ 의미와 같습니다.

7 🔴 분배 의심 Stage 2 + State 0·4 또는 다이버전스 전량 청산 검토. 손절 룰 작동.
🎯 왜 Phase 4가 핵심인가
Phase 4는 조기 진입 타이밍을 정량화한 유일한 Phase 입니다.

다른 매매 시스템들이 "이미 많이 올라간 Stage 2 리더"를 따라가는 반면, Phase 4는 Stage 2 "초기" + 단조 가속 (c1>c3>c6) + State 3·5·7 (건강한 모멘텀) 3중 조건을 모두 만족하는 순간을 잡습니다.

이는 오닐이 말한 "pivot buy point", 미너비니의 "VCP 돌파 시점"과 개념적으로 일치합니다. 차이는 이걸 감각이 아닌 숫자로 객관화했다는 것.

📈 Phase 전환 (오늘 막 바뀐 종목)

Phase 값 자체보다 "어제 대비 방금 Phase가 바뀌었는가"가 훨씬 강한 알파입니다. 특히 Phase 3 → Phase 4로 전환된 당일이 pivot buy point의 객관화된 순간 입니다.

🤔 자주 헷갈리는 질문

Q. Stage 2와 Phase 2 중 뭐가 더 좋아요?
둘은 완전 다른 축이라 비교 대상이 아닙니다. Stage 2 = 추세 살아있음 (좋음), Phase 2 = 바닥 매집 감지 (이른 후보). 이상적 조합은 "Stage 2 초기 + Phase 4" 또는 "Stage 1→2 전환 + Phase 3".
Q. Phase 4가 최고라면서 왜 Phase 5 종목이 더 많죠?
Phase 4 = "이제 막 돌파한" 순간적 상태라 짧게 머뭅니다. 대부분 며칠 내 Phase 5로 넘어갑니다. 그래서 Phase 4 종목 수는 항상 적고, 그것이 이 Phase가 진입 타이밍에 특화된 이유입니다.
Q. 이미 Phase 5인 종목은 사면 안 되나요?
살 수 있습니다. 다만 진입 타이밍으로는 Phase 4가 우위이고, Phase 5는 홀드·추가매수 구간입니다. Phase 5에서 신규 진입 시에는 기존 포지션 관리 목적이거나, 눌림 구간(풀백) 기다려 진입하는 것이 좋습니다.
Q. Phase가 없거나 "—"로 뜨는 종목은 왜죠?
RS Line 변화율 데이터가 부족할 때 발생합니다. 신규 상장주 (6주 이하), 데이터 공급 오류 종목 등. Phase 계산 최소 요구: 6주 이상의 가격 히스토리.
Q. MTR State 0~7을 일일이 외워야 하나요?
외울 필요 없습니다. 실전은 Phase 숫자만 보면 충분합니다. MTR State는 Phase가 왜 그렇게 나왔는지 근거 이해용이고, 특히 State 3 (전환 신호)와 State 7 (완전 상승) 두 개만 기억하면 됩니다.
Q. NEW↑ 배지가 붙은 종목이 갑자기 없어졌어요.
NEW 배지는 당일만 표시됩니다. 다음날이 되면 그 종목은 Phase 4 유지(2d)로 바뀝니다. 오늘 놓치면 그 "pivot buy point" 당일 기회는 놓친 것이지만, 며칠 유지된 Phase 4도 진입 가능 (Phase C 통계에서 Phase 4 1d > 2~3d > 4~7d 순으로 감소 확인).
Q. 어제 Phase 4였던 종목이 오늘 Phase 3으로 왜 떨어졌죠?
Phase 4 → Phase 3 은 하향 전환입니다. Phase 4 유지에 필요한 조건(S2 초기 + 단조 가속 + 양호 State)이 하나라도 깨지면 Phase 3로 내려갈 수 있습니다. 이는 돌파 실패 초기 신호로, 손절 룰에 가까워졌다는 뜻. 방향 배지 NEW↓ 붙은 것 확인.
Q. 다른 시스템(트레이딩뷰 등)에 Phase 개념이 없는데 이거 믿을 만해요?
Phase는 이 시스템의 독자 합성 지표입니다. 기본 원료(Stage, RS, 가속, 패턴)는 모두 오닐·미너비니의 고전 개념이므로 방법론 자체는 검증됨. 다만 Phase 숫자의 구체적 경계는 이 시스템만의 튜닝이므로, Phase C 백테스트로 30일+ 데이터 누적 후 본인 데이터로 확인하는 것이 원칙.
🎓 최종 요약 — 이것만 기억하세요
1. Stage와 Phase는 다른 축이다. Stage는 추세 상태, Phase는 액션 신호.
2. Phase 숫자 하나만 봐도 충분. Phase 4 = 진입 ★, Phase 5 = 홀드, Phase 6~7 = 익절·청산.
3. Phase 3 → Phase 4 전환 당일 (NEW↑) = pivot buy point의 객관화.
4. Phase 4+ (🎯+) = Phase 4 중에서 장기배경(6~10주)까지 양호한 최상위 후보 ★★★.
5. 매일 run_daily.bat 돌려야 전환 히스토리가 쌓임. 놓친 날은 데이터 공백.
6. Phase 4+ 종목이 적은 건 정상. 그게 이 필터의 최고 품질 희소성이다.
7. 30일+ 누적 후 백테스트 탭에서 Phase 4+ 알파 > Phase 4 알파 > 베이스라인 알파 확인 시 시스템 작동 증거.
